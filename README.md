# PONG

The first game `Pong` was developped by Atari in 1972.

![](pong.png)

[Usage](#usage) - [Releases](#releases) - [License](#license)

## Usage

```bash
# install the latest version
$ go install -v github.com/joakim-ribier/pong/cmd/client@latest

$ client
```
